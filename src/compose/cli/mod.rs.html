2# Compose CLI
3
4## Installation
5
6Install the latest version from GitHub using [Cargo](https://doc.rust-lang.org/cargo/):
7
8```bash
9cargo install --git https://github.com/Dutch-Raptor/compose.git
10```
11
12After installation, the `compose` executable will be available on your PATH:
13
14```bash
15compose --version
16```
17
18## Usage
19
20Create a new file called `hello.cmps` with the following contents:
21
22```text
23println("Hello, from Compose!")
24```
25
26and run it:
27
28```bash
29compose file hello.cmps
30```
31
32This should print `Hello, from Compose!` to the console.
33
34### Run a file
35
36```bash
37compose file hello.cmps
38```
39
40### Start a REPL
41
42```bash
43compose repl
44```
45
46Load a file in the REPL:
47
48```bash
49compose repl --from prelude.cmps
50```
51
52### Explain an error
53
54```bash
55compose explain E0003
56```
57
58This gives examples, explanations, and suggested fixes.
